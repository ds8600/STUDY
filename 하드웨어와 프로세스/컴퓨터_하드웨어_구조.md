# 컴퓨터 하드웨어 구조

## 1. 전체 구조(큰 그림)
- 컴퓨터는 보통 **CPU, 메모리(RAM), 저장장치(SSD/HDD), 입출력 장치**로 구성된다.
- 이들은 **버스(Bus)**를 통해 데이터를 주고받는다.

## 2. CPU
- 역할: 명령을 해석하고 실행하는 장치
- 구성(핵심):
  - **ALU**: 연산 담당
  - **CU(제어장치)**: 명령 해석/제어 신호 생성
  - **레지스터**: CPU 내부의 아주 빠른 저장공간

## 3. 메모리(RAM)
- 실행 중인 프로그램과 데이터를 올려두는 공간
- 특징: 빠르지만 전원이 꺼지면 사라짐(휘발성)

## 4. 저장장치(SSD/HDD)
- 프로그램/파일을 오래 저장하는 공간
- 특징: RAM보다 느리지만 영구 저장 가능(비휘발성)

## 5. 입출력 장치(I/O)
- 키보드/마우스/모니터/네트워크 카드 등
- 운영체제가 드라이버를 통해 제어한다.

## 6. 내 정리(예시 한 문장)
- “CPU가 실행하고, RAM이 작업대 역할을 하고, SSD는 창고 역할을 한다.”
